ASCII 코드 표

컴퓨터는 모든 문자를 숫자로 저장해요. ASCII 코드는 영어 알파벳, 숫자, 특수문자 등을 컴퓨터가 이해할 수 있는 숫자로 변환한 표입니다. 아래 표에서 자주 사용되는 문자들의 코드를 확인해보세요!

제어 문자 (0-31, 127)

제어 문자는 화면에 보이지 않는 특별한 명령을 내리는 문자예요. 예를 들어, 줄바꿈이나 탭 같은 기능을 수행합니다.

자주 사용되는 제어 문자:
  • 줄바꿈 (Enter): 새로운 줄로 이동
  • 탭 (Tab): 일정 간격 띄우기
  • 백스페이스: 이전 글자 지우기

출력 가능 문자 (32-126)

출력 가능 문자는 화면에 보이는 일반적인 문자들입니다. 영어 알파벳, 숫자, 특수문자 등이 여기에 포함됩니다.

자주 사용되는 문자:
  • 대문자 A-Z: 65-90
  • 소문자 a-z: 97-122
  • 숫자 0-9: 48-57

※ 주의사항
